## Project Overview

Ocean View Resort Portal is a web-based Room Reservation Management System developed to manage hotel bookings digitally. The system helps hotel staff handle reservations, avoid booking conflicts, and generate customer bills efficiently.

---

## Features

- User Authentication (Login System)
- Add New Reservation
- View Reservation Details
- Search Reservations
- Bill Calculation System
- Help & Support Section
- Secure Logout

---

## System Architecture

**Three-Tier Architecture**

- Frontend (User Interface)
- Web Server (HTTP Request Handling)
- Service Layer (Business Logic)
- Repository Layer (Data Management)
